CSS 기본 선택자

선택자(selector) 란? 특정한 HTML 요소(태그)를 선택할 때 사용하는 기능, 태그를 선택하여 원하는 스타일과 기능을 적용할 수 있다.

모든 선택자

HTML 문서 안의 모든 태그를 선택할 때 전체 선택자 사용 가능

  • 테스트1
  • 테스트2
  • 테스트3

태그 선택자

HTML 문서 내에 같은 이름을 가진 태그를 선택한다.

아이디 선택자

HTML 문서 내에 해당하는 ID속성을 가진 태그를 선택한다.

아이디를 나타내는 기호(#) - #아이디명{꾸밀값}

클래스 선택자

HTML 문서 내에서 동일한 클래스명을 가진 태그를 선택

클래스명을 나타내는 기호(.) ->.클래스명{바꿀값}
